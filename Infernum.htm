Description

Infernum is a German Style Strategy eBoardgame based on Greek Mythology. It is the revised, improvised and digitalized version of the board game “Battle for the Underworld” which was made during our First Year of college.

The game features Greek Gods and Greek Mythological creatures using which the players go head to head against each other in teams on a board consisting of hexagons. The game supports maximum of 4 players on the board. The game is deisgned to be a hot seat multiplayer game
